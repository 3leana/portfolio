# Portfolio Website

Personal portfolio showcasing my projects, skills, and experience as a software developer. Built with React, TypeScript, and Tailwind CSS.

**Live site:** _Coming soon_

## Features
- Smooth section-based navigation with active state highlighting  
- Fully responsive layout (desktop & mobile)  
- Technical project showcase with metrics and key highlights  
- Contact links (Email, GitHub, LinkedIn)

## Tech Stack
- **React** — UI components and state management  
- **TypeScript** — static typing and improved maintainability  
- **Tailwind CSS** — utility-first styling  
- **Vercel** — deployment and hosting  

## Deployment
The site is deployed using Vercel and configured for automatic builds on push.
